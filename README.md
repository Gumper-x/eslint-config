# @gumper-x/eslint-config

That basic eslint config for Vue 3 projects 

### <a href="https://www.npmjs.com/package/@gumper-x/eslint-config" target="_blank">NPM</a>

```bash
npm i @gumper-x/eslint-config -D
```
Use in `.eslintrc`
```js
{
  "extends": ["@gumper-x"],
  "rules": {
    // Your custom rules...
  }
}
```
